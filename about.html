This project is based on the concept of predicting real estate prices using machine learning. Inspired by Zillow's Zestimate, it uses data like bedrooms, bathrooms, and square footage to estimate property value.

Built using Python, ML algorithms, FastAPI, and MySQL — this tool helps home buyers and sellers get quick estimates.
